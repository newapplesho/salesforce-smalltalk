# salesforce-smalltalk
Salesforce API Library for Pharo Smalltalk

## Supported Smalltalk Versions

Pharo Smalltalk 5.0, 6.0

## Installation

```smalltalk
Metacello new
    baseline: 'Salesforce';
    repository: 'github://newapplesho/salesforce-smalltalk:v0.0.1/pharo-repository';
    load.
```
